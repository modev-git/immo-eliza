GET /predict Endpoint documentation

Returns a JSON object describing the expected input format for the POST /predict endpoint — useful for self-discovery without reading these docs.

POST /predict Predict a property price

Accepts property features as JSON and returns a predicted price in EUR. Missing optional fields are automatically imputed by the model.

Request body · application/json 
{
  "data": {
    "area": 120,                       // required
    "property-type": "HOUSE",        // required
    "rooms-number": 3,             // required
    "zip-code": 1050,               // required
    "land-area": 300,               // optional
    "garden": true,                 // optional
    "terrace": true,                // optional
    "facades-number": 4,            // optional
    "equipped-kitchen": true,       // optional
    "swimming-pool": false,         // optional
    "building-state": "GOOD"         // optional
  }
}
Response · 200 OK
{
  "prediction": 412500.0,
  "status_code": 200
}
